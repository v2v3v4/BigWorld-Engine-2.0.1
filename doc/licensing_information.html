BigWorld Technology External Libraries Licensing issues

The BigWorld Technology incorporates multiple external libraries with multiple licenses. In order to comply with the different licensing requirements of these libraries the following files are provided:

  • The development_and_deployment_licensing.txt file contains Licenses of external libraries included in the BigWorld server, server tools and client tools components. This file does not need to be redistributed to gamers.

  • The game_client_licensing.txt file contains licenses of external libraries included in the BigWorld game client. This license file should also be included in the documentation and/or other materials provided with any game which is created by BigWorld customers based on the BigWorld client. If Adboe Flash is not redistributed, the Adobe Flash section should be removed from this file before redistribution.

Flash Licensing Considerations

The BigWorld Technology Evaluation version includes the Adobe Flash Player component. This component is used to show flash content as part of the evaluation version. Redistribution of the Flash Player component is done based on a license coordinated with Adobe. Customers wishing to redistribute Adobe Flash Player should coordinate a similar license using http://www.adobe.com/products/players/fpsh_distribution1.html or instead make sure their installer links to a page allowing to install Flash Player for Firefox (plugin based). MySQL Licensing Considerations

Customers installing the server with a MySQL database should consider purchasing a MySQL license. Please review the licensing information at http://www.mysql.com/ for more details.

FMOD Licensing Considerations

BigWorld Technology includes the fmod audio library. Licensing considerations for this library should be coordinated with Firelight Technologies shortly before going live. See the fmod web site fmod for more details.
